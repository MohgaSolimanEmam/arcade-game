how to run the arcade game :
1-you can download it from (https://github.com/MohgaEmam/classic-arcade-game)
2-you unzip the file
3-you open your browser 
4-you open the (index.html) file and enjoy the game

how to play : 
press up : to move up .
press down: to move down.
press left: to move left.
press right: to move right.

the player will return to the originall point (200,400) at the following cases :
it collided with an enemy 
it reched the top of the screen
